Privacy Policy
1. Overview

Mood Manager is a demonstration web application built for educational and hackathon purposes.

2. Data Collection

Mood Manager does not collect, store, or share any personal data. All information entered into the predictor form is processed locally for the purpose of generating mood and stress predictions.

3. Tracking & Cookies

We do not use tracking technologies, analytics tools, cookies, advertising pixels, or third-party data services.

4. Data Storage

No user inputs are saved to a database or retained after prediction. All processing occurs in-session and is not stored.

5. Contact

As this is a demonstration project, there is no formal data controller.

Terms & Conditions
1. Use of the Application

Mood Manager provides predictive estimates of mood and stress levels based on lifestyle inputs. These predictions are generated using regression-based machine learning models.

2. Informational Purpose Only

The results provided by Mood Manager are for informational and demonstration purposes only. They are not medical, psychological, or professional advice.

3. No Liability

Mood Manager is not responsible for decisions made based on the predictions generated by the application.

4. No Data Retention

The application does not retain user input or track individual users. No personal data is collected or stored.

5. Modifications

As a hackathon project, features and functionality may change without notice.
